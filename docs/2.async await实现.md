### 1.1.1 返回 Promise 对象

async 函数如果使用 return 返回了值（如果没有 return 则返回 undefined），这个值会被 Promise.resolve() 包装成一个 Promise 对象。

```js
async function foo() {
  return 3;
}

foo().then(console.log);
// 3
```
