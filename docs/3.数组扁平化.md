# 1. 数组扁平化概念

数组扁平化是指将多维数组转换为一维数组的过程。在 JavaScript 中，我们经常需要处理嵌套的数组结构，将其转换为更易于操作的一维形式。

示例：

```javascript
// 扁平化前
const nestedArray = [1, [2, [3, [4]], 5]];

// 扁平化后
const flattenedArray = [1, 2, 3, 4, 5];
```

本文将详细介绍数组扁平化的概念和实现方法。
